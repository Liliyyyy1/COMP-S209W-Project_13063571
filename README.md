# Library Management System

## Description
This project is a simple OOP-based Library Management System written in Python. It simulates real-world library operations such as adding books, managing users, borrowing and returning books.

## Features
- Add book
- Add user (Student / Teacher / General)
- Show all books
- Show all users
- Borrow book
- Return book
- Show user's borrowed books
- Search book by title
- Remove book
- Count total books
- Count available books
- Count borrowed books

## OOP Concepts Used
- Class and Object
- Encapsulation (private attributes in Book class)
- Inheritance (StudentUser, TeacherUser inherit from User)
- Polymorphism (different borrow limits)
- Abstraction (menu-driven interaction)

## Files
- main.py: main program and user interface
- book.py: Book class
- user.py: User, StudentUser, TeacherUser classes
- library_system.py: system logic and functions
- README.md: project description

## How to Run
Run the following command in terminal:

python main.py

## Notes
This project is designed for academic purposes and demonstrates the use of Object-Oriented Programming concepts in Python.
